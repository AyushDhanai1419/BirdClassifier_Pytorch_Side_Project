Accuracy

Our model was able to achieve an accuracy of 94.5% on the test dataset.

Dataset

We've used Caltech-UCSD Birds 200 dataset for training and testing the model.

Transfer Learning

We've trained the model by using transfer learning on Densenet-121 convolutional neural network.

Hyper-parameters

Optimizer: ADAM
Loss Function: NLLLoss
LR Scheduler: StepLR
Batch Size: 90
Learning Rate: 0.001
